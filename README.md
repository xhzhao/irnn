# irnn


### Requirements
****
  * Pytorch
  * conda install mkl mkl-include
  * icc (if you only have gcc, please modify the CMakeList.txt line1 and line2)



### Installation
****
    mkdir build
    cd build
    cmake ..
    make
    cd ../pytorch_binding
    python setup.py install
